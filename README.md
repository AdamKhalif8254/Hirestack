## Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/yourusername/hirestack.git
   ```
2. Install dependencies
   ```bash
   npm install
   ```
3. Set up environment variables (see `.env.example`)
4. Initialize the database
   ```bash
   npm run db:push
   ```
5. Run the development server
   ```bash
   npm run dev
   ```
