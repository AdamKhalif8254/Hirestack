## Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/AdamKhalif8254/Hirestack.git
   cd Hirestack
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Set up environment variables
   ```bash
   npm install -g vercel
   vercel login
   vercel link
   vercel env pull .env
   ```

4. Run the development server
   ```bash
   npm run dev
   ```
